# Amazon Clone (MERN Stack)

Full-stack e-commerce application replicating core functionalities of Amazon.

## Tech Stack
- **Frontend:** React.js, Context API, Material UI
- **Backend:** Node.js, Express, MongoDB
- **Payments:** Stripe API

## Setup
1. `cd server && npm install`
2. `cd client && npm install`
3. `npm run dev`
